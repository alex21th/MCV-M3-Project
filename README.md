## Getting Started

### Prerequisites
Requires Python `>= 3.9`.

### Installation
#### 1. Clone the repo
```
git clone https://github.com/alex21th/MCV-M3-Project.git
```
#### 2. Create a virtual environment
```
python3 -m venv /path/to/new/virtual/environment
```
#### 3. Install package requirements
```
pip install -r requirements.txt
```
